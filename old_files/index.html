Virtual Twenty-Four Game

Github Link
C

There are two versions of this text-based virtual implementation of the Twenty-Four Game, a game where you have 4 numbers, and using mathematical operators, have to try and come up with a way to make an equation that equates to 24. Each version focuses on a different aspect of coding.

First Version
  • Simpler version of the game, simply input the operators in the order you want them used.
  • Implemented features such as easy mode, debug mode, and operator input validation.
  • Generated and stored a set of 3188 solutions using a brute-force algorithm for debugging and hard mode.
Second Version
  • More advanced version, input the full expression you want used, can include parenthesis as well.
  • Focuses on order of operations with parenthesis, stack implementation with linked lists (pop, isEmpty, push, peek, etc.), run-time, and preventing memory leaks.
  • Implemented a debug mode and designed an algorithm that found all possible solutions to the game.
  • Includes lots of error handling for matching parenthesis, correct numbers used, all numbers used, valid operators, etc.
